**Task 2: Chatbot for FAQs
**


😊 Smile AI Assistant
A smart FAQ Chatbot built with Python, Streamlit, and NLP. It uses TF-IDF and Cosine Similarity to understand user questions and replies via a modern dark themed UI.

Features:-

Smart Matching: Uses TfidfVectorizer and cosine_similarity for intent matching.
Modern UI: Clean Glassmorphism dark theme using st.chat_message.
Safety Filter: Inbuilt profanity block and out of context fallback.
Exit Control: Typing exit or quit safely closes the session with a restart option.

Tech Stack:-

Python 3.13 plus
Streamlit, NLTK, Scikit-Learn
Quick Start

Install: pip install streamlit nltk scikit-learn

Run: c:/python313/python.exe -m streamlit run "c:/Users/user/Downloads/CHATBOT FAQs.py"
